SIANNESTRANSPORTSERVICES

The web application is designed to automate and improve the speed and accuracy of recording, monitoring, and retrieving students pick-up and drop-off records. It is intended for students and parents, used to track and record students' entry and exit from the van. Each scan will automatically send an email to the guardian with the time the student entered and exited the van.

FEATURES

DASHBOARD

The dashboard is a key tool for administrators to manage users, vans, and students, displaying total counts and a detailed table of students' entry and exit times. It allows printing and downloading records as an Excel file. Helpers/drivers can scan QR codes to automatically email students' contacts for timely communication and safety.

STUDENTS MASTERLIST

is a comprehensive record containing detailed information for each student, ensuring efficient management, accessibility, and up-to-date records. It features a user-friendly interface to add new students, print QR codes, remove students with an option to view removed entries, and update personal information as needed.

SCANNER

A scanner feature enables helpers/drivers to scan individual QR codes, automatically sending an email to the student's contact person to ensure timely communication and enhanced safety.

VAN SERVICE

The School Service section allows administrators to manage the school's transportation services. Here, you can add new school services and remove existing ones, ensuring efficient organization and tracking of all transportation options available to students. This feature helps maintain an up-to-date record of all active school services.

USERS CONFIGURATION

allows administrators to manage user accounts. Here, you can add, remove, and update users. There are two types of users: Admin and Helper/Driver. Admins have full access to all features and settings within the system, while helpers or drivers have restricted access, limited to the dashboard that includes time in and out records and the scanner. This configuration ensures appropriate access control and efficient user management.

ACTIVITY LOGS

The Activity Log provides a comprehensive record of all changes made within the system. This includes actions such as adding vans, removing students, and other modifications. It allows administrators to monitor and review all activities to ensure accountability and track system updates effectively.
