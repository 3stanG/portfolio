GMA MUNICIPAL JAIL ID SYSTEM

Is a web application that is intended to implement an automated system that enhances the speed and accuracy of record, monitoring, and retrieving record of visitors

FEATURES

DASHBOARD

Provides insights into visitor entry times, active/inactive ID counts, and PDL statistics. Allows analysis of visitor trends by date selection for both relative and non-relative visitors

VISITORS LOGS

Utilizing our advanced scanner technology, this feature diligently records entry and exit times alongside key data. Effortlessly track visitor activity for streamlined security and effective facility management

SCANNER

MJIS provides two scanning options: webcam or QR machine, additionaly a manual input for control numbers. Quickly access detailed ID information for efficient visitor management

RELATIVE VISITORS

Facilitates the addition and tracking of relative visitors, allowing for ID printing. Upon addition, the module requests relevant visitor information for comprehensive tracking

NON-RELATIVE VISITORS

Adds and tracks non-relative visitors under three categories: general use, service provider, and contractor. Includes ID printing for easy identification without additional visitor information."

USERS

Seamlessly handles user management tasks, including the addition of new users, editing existing user information, and assigning access levels tailored to specific user requirements.

PDL

Simplifies PDL management by allowing easy addition, removal of PDLs upon availability, and editing of information. Includes a convenient import feature for adding multiple PDLs efficiently.

ACTIVITY LOGS

Tracks system data changes, such as additions of new PDLs, visitors and etc. It displays the user, time, and date details of the user. Provides valuable insights for system monitoring and tracking

PROFILE

Enables users to update their personal information within their designated ranks, with administrative access reserved for the admin. Maintaining security and control over sensitive data
